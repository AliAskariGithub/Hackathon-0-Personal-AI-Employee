# Company Handbook

## Core Operating Rules

1. **Always prioritize safety.** Ensure all operations are secure and do not compromise system integrity or data privacy.

2. **Move files only when complete.** Files should remain in their current location until all processing is finished and verified.

3. **Use professional tone.** All communications and documentation must maintain a professional, clear, and respectful tone.

---

## Workflow Guidelines

- **Inbox**: New items requiring initial review
- **Needs_Action**: Items identified for processing or action
- **Done**: Completed items archived for reference
